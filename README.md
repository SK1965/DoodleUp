# DoodleUp

Real-time collaborative drawing application.

## Features
- Real-time drawing synchronization across users
- Join existing drawing rooms
- Minimalist and aesthetic glassmorphism UI
- Canvas tools: pen, eraser, color picker, size adjust
- Notification system for room activities
- Export canvas as PNG

## Tech Stack
- Frontend: React, Vite, Tailwind CSS
- Backend: Node.js, Express, Socket.io
- Real-time Communication: WebSockets

## Getting Started

1. Clone the repository
2. Install frontend dependencies: `cd client && npm install`
3. Install backend dependencies: `cd server && npm install`
4. Start development servers: run `npm run dev` in both directories.
